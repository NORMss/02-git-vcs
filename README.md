# Project 02-git-vcs

### Files will be [ignored](./.gitignore):

1. Compiled class file
```
*.class
```
2. Log file
```
*.log
```
3. BlueJ files
```
*.ctxt
```
4. Mobile Tools for Java (J2ME)
```
.mtj.tmp/
```

5. Package Files
```
*.jar
*.war
*.nar

*.ear
*.zip
*.tar.gz
*.rar
```
5. virtual machine crash logs, see http://www.java.com/en/download/help/error_hotspot.xml
```
hs_err_pid*
replay_pid*
```
